## PuTTY - About

PuTTY is great because it makes it much easier to paste in values, versus other terminal emulators. Get it [here](https://putty.org/).

<br/>

## <span style="color: #00ff7f">How to Install Themes (from .reg files)</span>

PuTTY changes colors based on config files that you have to add to your registry. It's easier than it sounds:

1. Find a [.reg](https://github.com/AlexAkulov/putty-color-themes/tree/master) file you want to use (more links below)
2. Paste the content of the .reg file into your text editor (like Notepad++ or Notepad). Change the "Default%20Settings" text string unless you want this theme to be your default:

![2.png](/images/putty/2.png)

3. Save this .reg file anywhere
4. Double click on the .reg file, click "Yes" to add the .reg file to the registry
